# DINIzawrus Ruller Bot

Discord-бот для модерации, правил сервера, музыки в голосовом канале и простых развлекательных команд.

## Возможности

- Модерация: варны, муты, баны, правила сервера
- Slash-команды: правила, публикация правил, музыка и управление воспроизведением
- Музыка: поиск треков на YouTube, выбор из 5 вариантов, пауза/воспроизведение, следующий/предыдущий, стоп, громкость, loop
- Развлечения: случайное число (`?ролл`), монетка (`?монетка`)

## Команды

### Модерация
- `?правила`
- `?правила чата`
- `?правила войса`
- `?правилапост [#канал]`
- `?варн @user`
- `?снятьварн @user`
- `?варны @user`
- `?мут @user [5мин|2час|1дн]`
- `?размут @user`
- `?муты`
- `?бан @user`
- `?разбан <id>`
- `?баны`

### Музыка
- `?плей <запрос>`
- `?1` — `?5` для выбора трека из поиска
- `?пауза`
- `?унпауза`
- `?луп`
- `?некст`
- `?пред`
- `?стоп`
- `?громкость 1-100`

### Развлечения
- `?ролл`
- `?ролл 1-100`
- `?монетка`
